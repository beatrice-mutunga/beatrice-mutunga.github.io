Exploring real-world cybersecurity and CTF problem-solving

Challenge 1: SQL Injection Discovery

Problem Statement

Find and exploit a SQL injection vulnerability on a sample login page to retrieve hidden user data.

Approach

Tested different input parameters using Burp Suite and manual payloads. Identified a vulnerable POST parameter that revealed database errors.

Tools Used

Burp Suite, SQLMap, Browser DevTools

Key Lessons Learned

Gained experience in identifying injection points, crafting payloads, and automating the process with SQLMap.

Challenge 2: Web Exploitation – File Upload Bypass

Problem Statement

Bypass a file upload filter to execute a PHP reverse shell.

Approach

Analyzed server responses and crafted a disguised PHP payload with image headers. Uploaded using intercepted requests.

Tools Used

Burp Suite, Kali Linux, netcat

Key Lessons Learned

Learned about file upload sanitization, MIME-type validation, and how to avoid common web app misconfigurations.
